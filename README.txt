This is the first part of the capstone project for the HarvardX PH125.9x course.

Contents:

	- movielens.RData = R script for the project
	- movielens_report.Rmd = RMarkdown script for the report
	- movielens_report.pdf = PDF version of the report
	- movielens.RData = movielens data table, to avoid re-creating every time the scripts need to be run
	- edx.RData and validation.RData = training set (edx) and test set (validation), subset from the movielens data
	- rmse_results.RData = final table of RMSE results, used for the report
